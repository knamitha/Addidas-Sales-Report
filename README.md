# Adidas-Sales-Report
A data analysis project focused on exploring Adidas sales data to identify trends, performance insights, and key business metrics using data analytics and visualization techniques.

## Project Overview
This project focuses on analyzing Adidas sales data to uncover trends, regional performance, and profitability. 
Using Tableau, the project transforms raw sales data into interactive visual dashboards that support data-driven business decision making.

## Objectives
- Analyze overall sales and operating profit of Adidas products
- Identify high-performing regions and retailers
- Study sales trends over time
- Compare different sales methods
- Create an interactive Tableau dashboard for business insights

## Dataset Information
- Dataset Name: Adidas Sales Dataset
- Format: CSV
- Domain: Retail & Sales Analytics
- Data Type: Structured
- Source: Kaggle

## Dataset Features
- Date
- Product
- Region
- Retailer
- Sales Method
- Price per Unit
- Units Sold
- Total Sales
- Operating Profit
- Operating Margin

## Data Preprocessing
- Cleaned numerical fields by removing currency symbols and commas
- Converted sales and profit fields to numeric format
- Categorized fields into measures and dimensions in Tableau
- Formatted date fields for time-series analysis
- Created calculated fields for sales and profit analysis

## Tools & Technologies
- Tableau – Data visualization and dashboard creation
- CSV Dataset – Data source

## Key Performance Indicators (KPIs)
- Total Sales
- Total Operating Profit
- Units Sold
- Average Price per Unit
- Operating Margin

## Dashboard Overview
The Tableau dashboard provides an interactive view of Adidas sales performance. 
Users can filter data by region, product, retailer, and sales method to analyze trends, profitability, and regional contribution.

## Key Insights
- Certain regions contribute significantly higher sales and profit
- Online and outlet sales methods show strong performance
- Some products generate high sales volume but lower profit margins
- Clear seasonal sales trends are observed

## Business Impact
- Supports data-driven decision making
- Helps identify profitable regions and products
- Improves understanding of sales performance
- Enables better strategic planning

## Limitations
- Dataset is limited to historical sales data
- External factors such as promotions and competition are not included

## Future Enhancements
